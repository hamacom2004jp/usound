Command ReferenceÔÉÅ

See below for available options for the various commands.

  • Speaker
    • Capture speaker audio : usound -m speaker -c capture <Option>
    • List speaker : usound -m speaker -c list <Option>
  • Whisper
    • Whisper model deploy : usound -m whisper -c deploy <Option>
    • Whisper model start : usound -m whisper -c start <Option>
    • Whisper model stop : usound -m whisper -c stop <Option>
    • Whisper model transcribe : usound -m whisper -c transcribe <Option>
  • Install
    • Installation (server) : usound -m install -c server <Option>
